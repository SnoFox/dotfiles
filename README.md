# SnoDots NG
Algabraic, simplified, modernized dotfiles for SnoFox

## Using them
1. Install chezmoi
    * macOS: `brew install chezmoi`
    * Linux: `sh -c "$(curl -fsLS get.chezmoi.io)" -- -b ~/.local/bin`
2. `chezmoi init SnoFox`
3. `chezmoi diff`
4. `chezmoi apply` # if happy with the diff
